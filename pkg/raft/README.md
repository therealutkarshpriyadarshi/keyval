# Raft Package

Core Raft consensus algorithm implementation.

## Components

- **node.go**: Raft node structure and state management
- **election.go**: Leader election logic
- **replication.go**: Log replication
- **log.go**: Log management
- **snapshot.go**: Snapshot creation and installation
- **rpc.go**: RPC definitions and handlers
- **persistence.go**: Persistence layer integration
